# WorldCup Copilot

WorldCup Copilot 是一个面向体育赛事内容生产的 AI 工作台。它不是普通写稿器，而是围绕一场足球比赛，完成“赛事数据 → 选题角度 → 图表解读 → 多平台内容 → 风险审稿 → 方案报告”的完整工作流。

## 目标用户

- 体育记者、编辑和解说内容团队
- 自媒体创作者、B站 UP 主、短视频创作者
- 小红书、微博、公众号等平台内容运营
- 赛事运营、品牌内容和体育营销团队

## 解决痛点

- 赛后信息密度高，选题容易同质化。
- 数据图表和文字解读分离，内容生产效率低。
- 不同平台内容风格差异大，手动改写成本高。
- 体育内容容易出现伤病、判罚、引战、数据来源等发布风险。
- 运营主管需要完整方案，而不是零散标题和文案。

## 核心功能

- 首页 Dashboard：展示产品定位、流程和示例比赛。
- 赛事输入：支持示例数据、手动输入、CSV / JSON 粘贴。
- 选题引擎：生成差异化选题，并评估平台适配、制作难度和风险等级。
- 数据洞察：展示控球率、射门 / 射正、关键球员雷达图、事件时间线和历史交锋。
- 内容工坊：生成 B站、小红书、微博、短视频、公众号 / 专栏内容。
- 知识库问答：使用本地 mock 知识库回答世界杯相关问题。
- 风险审稿：提示伤病、判罚、引战、侮辱、事实观点混淆和数据来源风险。
- 方案报告：生成可复制、可导出的 Markdown 内容运营方案。
- 数据说明：说明数据来源、隐私边界和合规使用方式。

## 数据来源说明

当前版本默认接入 2026 世界杯免费公开数据源，服务端会优先请求 WorldCup26 免费 API 获取赛程、比分、分组、球队和场馆信息；当该源不可用时，回退到 TheStatsAPI 免费赛程静态数据。所有外部数据都会在服务端转成项目内部统一结构，前端不会直接依赖第三方原始字段。

免费公开源通常不包含完整 xG、射门分布、传球线路、球员跑动等深度技术统计。遇到缺失字段时，系统会保留比赛基础信息，并在数据洞察与内容生成里按“数据覆盖有限”处理。

本项目仍保留本地示例数据、用户输入数据和手动导入数据作为 fallback。所有示例数据都在 UI 中标注“示例数据”。

当前版本不后台抓取抖音、小红书、B站、微博等平台登录内容，不保存账号、密码、Cookie 或个人平台会话。正式版本可接入授权体育数据 API、公开数据集或机构自有数据源。

## AI 使用方式

当前版本不依赖外部 API Key，AI 相关能力以本地 mock 规则和模板模拟，确保本地环境可以完整演示。

可替换或扩展的模块：

- `/data`：示例比赛数据和本地知识库。
- `/lib/ai/topics.ts`：选题生成逻辑。
- `/lib/ai/content.ts`：多平台内容生成逻辑。
- `/lib/ai/risk.ts`：风险审稿规则。
- `/lib/ai/report.ts`：方案报告生成逻辑。

## 本地运行方式

```bash
npm install
npm run dev
```

可选环境变量：

```bash
# 默认值就是 free-2026，不需要 API key
WORLD_CUP_DATA_SOURCE=free-2026

# 如需改回 API-Football，可配置：
WORLD_CUP_DATA_SOURCE=api-football
API_FOOTBALL_KEY=你的服务端 key
API_FOOTBALL_SEASON=2026

# 如需启用 DeepSeek 增强内容推理：
DEEPSEEK_API_KEY=你的 DeepSeek key
DEEPSEEK_MODEL=deepseek-v4-flash
```

默认地址：

```text
http://localhost:3000
```

指定端口示例：

```bash
npm run dev -- -p 3002
```

## 5 分钟演示路径

1. 打开首页，点击“使用世界杯示例数据演示”。
2. 在赛事输入页确认示例比赛为“2022 世界杯决赛：阿根廷 vs 法国”。
3. 点击“生成选题”，进入选题引擎。
4. 选择“梅西职业生涯最后拼图”。
5. 进入数据洞察页，查看控球率、射门 / 射正、雷达图、时间线和异常提示。
6. 进入内容工坊，查看 B站、小红书、微博、短视频和公众号内容。
7. 进入风险审稿，使用风险样例检查发布风险。
8. 进入方案报告，复制或导出 Markdown。

## 合规说明

- AI 生成内容仅作为创作辅助，不替代人工编辑和事实核查。
- 涉及伤病、判罚、争议、数据来源时，应补充可靠来源并人工确认。
- 不建议使用“裁判黑哨”“靠黑幕夺冠”“确认伤退”等定性表达。
- 正式发布前需要检查事实、数据来源、素材版权和平台规则。
