# Apple 风格卡片 Hover 设计

## 目标

统一主要卡片的 hover 反馈，使其轻微浮起、材质提亮、层级更清楚，同时保持现有布局、业务逻辑、数据结构和配色体系不变。

## 动效

- 位移：向上 2px。
- 缩放：1.01。
- 时长：240ms。
- 缓动：`cubic-bezier(0.22, 1, 0.36, 1)`。
- 属性：仅过渡 transform、box-shadow、border-color 和 background-color。
- 禁止旋转、弹跳、闪烁、粒子和大幅缩放。

## 材质

- 深色卡片：白色低透明边框、深青玻璃底、柔和深色阴影。
- 浅色卡片：足球绿低透明边框、接近白色的清透背景、柔和蓝灰阴影。
- 强调卡片：使用淡金边框变量，不增加发光。

## 接入范围

- 首页赛事卡片。
- 首页热点卡片。
- 比赛详情页信号、热点、平台和审稿卡片。
- 数据图表卡片。
- 热点详情页 Panel 和指标卡片。
- 深色工具页复用的共享 Card 及主要重复功能卡片。

页面级 section、表单控件、按钮、弹窗和图表内部元素不接入。

## 无障碍

- 仅在支持 hover 且精细指针设备上执行位移和缩放。
- `prefers-reduced-motion: reduce` 下沿用项目现有的全局降级规则。
